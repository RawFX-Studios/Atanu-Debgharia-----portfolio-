# Atanu Debgharia Portfolio Website

A simple static portfolio website for a video editor and motion graphics designer.

## Files

- `index.html` — main landing page and site structure.
- `styles.css` — site styles and layout.
- `script.js` — page interaction logic.
- `assets/` — image and video assets used by the site.

## Description

This project features:

- hero video background
- glassmorphic navigation
- About, Services, Skills, Projects, and Contact sections
- responsive, static HTML/CSS layout

## How to use

1. Open `index.html` in your browser.
2. Edit text, images, or video sources as needed.
3. Update `styles.css` for custom styling.
4. Add more interactivity in `script.js`.

## Deployment

Because this is a static website, you can deploy it to any static hosting service such as GitHub Pages, Netlify, or Vercel.
